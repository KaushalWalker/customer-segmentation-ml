# Telecom Customer Churn Prediction

This project predicts whether a telecom customer is at risk of churning using a machine learning model.  
The model analyzes customer features such as tenure, monthly charges, and contract type to estimate churn risk.

## Live Application
https://customer-segmentation-ml-5af3encwh2beacpsny3bj8.streamlit.app/

## Project Overview
Customer churn is a major challenge for telecom companies.  
This project uses machine learning to identify customers who are likely to leave the service.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- numpy
- Streamlit

## Features
- Interactive web interface for customer input
- Real-time churn risk prediction
- Simple ML model deployment
- User-friendly dashboard

## Project Workflow
1. Data preprocessing and feature selection
2. Model training using Scikit-learn
3. Model saving using Joblib
4. Web application development using Streamlit
5. Deployment using Streamlit Cloud
